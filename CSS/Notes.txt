CSS(Cascading stylesheet):
 -CSS is an style language used for describing the presentation of a document written in a mark up language(HTML).
 -CSS is used to add style to a webpage.

 -There are 3 ways to add style or CSS to a webpage-
 1.Inline CSS
 2.External CSS
 3.Internal CSS

 Inline CSS:-
 -Adding style for an individual element with the help of style attribute.
 
 syntax:
 <p style="font-size:30px; font=weight:70;>

 Internal CSS:-
 -used to add style multiple or writting in general style with the help of style tag inside head tag.
 -we need selector to target or select the element which we need to change type 

 syntax:
 <style>
    .className{
        font-size:20px;

    }
    #idName{
        background-color:red;
    }
</style>

External CSS:-
-writing css in separate css file and then connecting it to the html document with the help of link tag
-We used link tag inside the head tag to connect the external css.

syntax:
<link rel="stylesheet" href="./style.css">

.app{
    background-color:pink;
    font-size: small;
}

SELECTORS IN CSS:

-CSS selectors are used to select the html elements that we want tp change style.
-There are 6 types of selectors in css:
1.Universal selectors
- It is used to select all the elements in html document.
-It is denoted by *

*{
    font-size:80px;
}

2.Id selector
- It select the html element based on the given id name.
-Id name can be provided by using"id" attribute.
-Id name should be unique in an html document.
-To select the id name in css we use # which acts as id selectors.

3.class selector:
-It select the html element based on the given class name.
-class name can be provided by using "class" attribute.
-class name can be duplicate.
-To select the class name in css we use . that acts as class selector.

4.Element selector:
-It is used to select the elements based on their tag name.
-Element selectors selects all the element with the specific tag name in the entire html document.

p{
    background-color:red;
}

5.Descendent selector:
-It is used to select a specific child element with in a parent element.

6.Group selector:
-It is used to select multiple elements in the html document.


Felx Box:-
-This property is used to structure the elements in row wise or column wise by giving space allocation for individual elements within the flex area.
-Flex property is used in parent container to structure child element either in row or column
Propertis of flex:-
1.Flex-direction:-It specify from which direction the elements should be aligned.
  /* flex-direction: row; */
    /* flex-direction: row-reverse; */
    /* flex-direction: column; */
    /* flex-direction: column-reverse; */
2.Justify-content:-This is used to allign the elements horizontally in the flex area.
 /* justify-content: start; */
    /* justify-content: center; */
    /* justify-content: flex-end; */
    /* justify-content: space-between; */
    /* justify-content: space-around; */
    /* justify-content: space-evenly; */
3.Align-items:-It is used to align the items vertically in the flex area.
/* align-items: end; */
    /* align-items: center; */
    /* align-items:start; */
 4.Flex-wrap:-It says whether flex items are forced into one line or wrap into multiple line.
    flex-wrap:wrap;
    flex-wrap:no wrap;
 5.Gap:-It is used to provide gap between child elements inside a parent container.
    gap:20px;

    CSS Units:-
    1.Pixel(px):-
         It is fixed for each and every screen.
         1px=1/96inch.
    2.%:-
        It fully dependes upon their parent element.
    3.vh/vw(viewport height and viewportwidth):-
        It depends the size as per screen size(viewport)
    4.vmax:-
        It compares between vh and vw.
        Which value will be greater 
    5.em/rem:-
        They are relative length units used for sizing elemnets particularly for typography.
        em unit is relative to font size of parent element.
        rem unit is relative to font size of root element/default font size.
        1rem=16px.


        Transform Properties:-
          These are used to transform or change an element from one from to another form or to move from one position to another position.

          1.Rotate:-
             It is used to rotate an object in clockwise or anti-clockwise direction.
             To rotate an element we use degree(deg) as an unit.
                             /* transform: rotateX(60deg); */
                             /* transform: rotateY(60deg); */
                             /* transform: rotateZ(60deg); */
                             /* transform: rotate(50deg); */ 

          2.Skew:-
              It distorts point of an element by a certain angle in horizontal or vertical direction.

              Properties:-
              1.For horizontal skewx.
              2.For vertical SkewY.

          3.Scale:-
              It allows us to resize an element compared to its original size.
              Scale less than 1 will decrease the size and scale more than 1 will increase the size of an element.

              Properties:-
              1.ScaleX:-It will change the size in X axis or horizontally.
              2.ScaleY:-It will change the size in Y axis or vertically.
              3.Scale:-It will change the size in all direction.

          4.Translate:-
              It allows us to change the position of an element from its original position to another position along the X and Y axis.

              Properties:-
              1.TranslateX:It Translate the element over X axis from left to right or right to left.
              2.TranslateY:It translate the element over Y axis from top to bottom or bottom to top.
              3.Translate(value 1, value2):It translate the element in both the axis value 1 for X axis and Value2 for Y axis

          5.Transition:-

               Transition enable us to refine the transition between two state of an element smoothly.

               Properties:-
               1.transition-property: transform;:-Property for which you want to add transition.Example:font size,background color,all,etc.
               2.transition-duration:Time to change from one state to another state.Example 1s.
               3.Transition-timing-function:To control the speed.ease-in,ease-out,ease-in-out,linear,steps.
               4.transition-delay:Delay to start the transition.Example:1s.
               Or in short we can write:
                   transition:propertyname duration timing-function delay;
